# CaseDesk

CaseDesk is a learning-first full-stack legal workflow management app built from scratch to deepen practical software engineering skills.

## Goals
- Build a real full-stack system from zero
- Master ASP.NET Core fundamentals
- Learn EF Core with PostgreSQL
- Practice React frontend integration
- Use Docker and GitHub Actions
- Build disciplined Git and project habits

## Planned Stack
- ASP.NET Core Web API
- EF Core
- PostgreSQL
- React + TypeScript
- Docker
- GitHub Actions

## Localization
- English first
- Turkish later
- German in the future
